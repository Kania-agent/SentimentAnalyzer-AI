# SentimentAnalyzer-AI

**AI-Powered Sentiment Analysis & Social Monitoring**

SentimentAnalyzer-AI is a social media themed sentiment analysis platform built with MiMo V2.5 intelligence. It provides real-time text analysis, sentiment gauges, trend charts, and multi-platform monitoring.

## Features

- **Text Input Analysis** — Paste text or enter comments for instant sentiment scoring
- **Sentiment Gauge** — Visual semi-circular gauge showing positive/negative/neutral breakdown
- **Trend Chart** — Track sentiment over time with CSS-based visualization
- **Social Feed** — Simulated social media posts with live sentiment tagging

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- No external dependencies
- Social media theme with modern UI

## Getting Started

Open `index.html` in any modern browser to start analyzing sentiment.
